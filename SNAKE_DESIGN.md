# Snake Hookah — дизайн-система

> Файл-контекст для Claude Code. Тримати в корені репозиторію.
> Всі кольори взято піксельним семплінгом з бренд-зображень, не «на око».

---

## Правила

1. **Ніяких хардкод-кольорів у компонентах.** Тільки `var(--*)`. Знайшов `#fff` у JSX — виніс у токен.
2. **Один акцент на екран.** Золото (`--gold`) — це заклик до дії. Два золотих елементи поруч = жоден не працює.
3. **Кольори колекцій ≠ кольори UI.** Зелений/теракота/кава живуть у картках товару як мітки лінійки. Кнопку в них не фарбуємо.
4. **Темна тема — база, не опція.** Світлої версії немає. Не додавати.
5. **Мінімум радіусів.** 8px — картки, 3px — бейджі, 100px — чіпи. Все інше — 0.

## Pantone (для друку)

| Токен | HEX | Pantone C | Де |
|---|---|---|---|
| `--gold` | `#C9A961` | **871 C** (metallic) / **4515 C** | Тиснення на шкірі, фольга на коробці |
| `--ink` | `#0E0E10` | **Black 6 C** | Коробка, пакування |
| `--snake-bg` | `#595959` | **Cool Gray 11 C** | Ілюстрація |

## Фони секцій

П'ять кольорів з бренд-ілюстрацій. Це палітра для чергування секцій на довгих сторінках — щоб сторінка не була однорідно чорною від початку до кінця.

```css
:root {
  --sec-graphite: #595959;  /* сірий — з ілюстрації як є */
  --sec-forest:   #2A3C3C;  /* хвойний — з ілюстрації як є */
  --sec-stone:    #BFB9B9;  /* світлий камінь — ЄДИНИЙ світлий фон */
  --sec-brick:    #743621;  /* теракота — з ілюстрації як є */
  --sec-coffee:   #523C2A;  /* кава, затемнена (див. нижче) */
}
```

**Пари текст/фон — перевірено розрахунком WCAG, не на око:**

| Фон | Текст | Приглушений | Акцент | Ratio |
|---|---|---|---|---|
| `--sec-graphite` | `--text` #E8E6E3 | `#C4C1BD` (3.9 — 18px+) | **`--text`**, не gold | 5.6:1 |
| `--sec-forest` | `--text` #E8E6E3 | `#A8B0AE` (5.2) | `--gold` (5.2) | 9.3:1 |
| `--sec-stone` | **`--ink` #0E0E10** | `#3A3835` (6.0) | `--ink` | 10.0:1 |
| `--sec-brick` | `--text` #E8E6E3 | `#E0C4B8` (5.6) | `#E8C87A` | 7.4:1 |
| `--sec-coffee` | `--text` #E8E6E3 | `#C9B8A8` | `--gold` (4.6) | 8.3:1 |

**Дві поправки до вихідних кольорів ілюстрацій:**

1. **`--sec-coffee` затемнено з `#8B694E` до `#523C2A`.** Оригінал дає 4.0:1 зі світлим текстом і 3.9:1 з темним — провалює AA в обидва боки. Це «мертвий» середній тон, на якому не читається нічого. Затемнений варіант дає 8.3:1 і пропускає золото.

2. **`--sec-stone` — інвертувати текст на `--ink`.** Світлий текст дає 1.6:1. Це єдина секція з темним текстом на сайті.

**Золото не працює на `--sec-graphite`** (3.1:1 — тільки для 18px+/bold). Акцент там — звичайний `--text`.

**Правила:**
- `--ink` — база. Кольорові секції — акценти, не більшість сторінки.
- Не більше 2–3 кольорових секцій на сторінку.
- Два кольорові фони поспіль — ні. Між ними завжди `--ink`.
- `--sec-stone` — найрізкіший перехід на темному сайті. Один раз на сторінку, максимум.

## Компоненти

### Кнопки
```css
.btn { padding: 15px; border-radius: var(--r-md); font-size: var(--t-sm);
       font-weight: 600; border: none; cursor: pointer; transition: var(--fast); }

.btn-primary   { background: var(--gold); color: var(--ink); }
.btn-primary:hover { background: var(--gold-hi); }
.btn-secondary { background: transparent; color: var(--text); border: 1px solid #3A3A3F; }
.btn-secondary:hover { border-color: var(--gold); color: var(--gold); }
.btn:disabled  { background: var(--gold-dim); cursor: not-allowed; }

:focus-visible { outline: 2px solid var(--gold); outline-offset: 2px; }
```

Одна `.btn-primary` на екран. У картці товару це «Купити»; «Купити в 1 клік» — завжди secondary.

### Ціна
```css
.price { font-family: var(--font-num); font-variant-numeric: tabular-nums;
         font-size: 30px; font-weight: 600; color: var(--gold); }
.price small { font-size: var(--t-sm); color: var(--text-mute); font-weight: 400; }
```
Гривня — основна, долар у `small`. Не навпаки: клієнт платить у гривні.

### Бейдж лінійки
```css
.badge { font-size: var(--t-label); letter-spacing: .14em; text-transform: uppercase;
         color: var(--gold); border: 1px solid var(--gold);
         border-radius: var(--r-sm); padding: 4px 10px; }
```

### Наявність
```css
.stock { font-size: var(--t-xs); color: var(--ok); }
.stock::before { content: '●'; margin-right: 6px; }
.stock--out { color: var(--err); }
```

## Фото товару

Всі 11 знято на **білому** тлі. На `--ink` це вирізає яскравий прямокутник.

```css
.gallery { background: linear-gradient(160deg, var(--ink-3), #121214);
           border: 1px solid var(--line); border-radius: var(--r-lg); padding: var(--s-5); }
.gallery img { border-radius: var(--r-md); background: #fff; }
```

Рамка + падінг гасять контраст. **Не** ставити білі фото прямо на `--ink` без обгортки. Довгостроково — зняти каталог на `--ink` або вирізати фон.

## Мова інтерфейсу

- Українська. Кнопки — дієслово в інфінітиві: «Купити», «Додати в кошик», «Оформити замовлення».
- Дія зберігає назву на всьому шляху: кнопка «Купити» → тост «Додано в кошик» → сторінка «Кошик».
- Ціна завжди в гривні, курс перераховується на бекенді. Не показувати `$285` без гривні поруч.
- Помилка каже, що робити: «Товар закінчився. Повідомимо, коли з'явиться» — не «Помилка 404».
- Порожній кошик — запрошення, не констатація: «У кошику порожньо. Подивитись Spiral →»

---

## Що не робити

- ❌ Не додавати другий акцентний колір. Золото одне.
- ❌ Не заливати картку кольором лінійки. Тільки смуга/крапка.
- ❌ Не робити світлу тему.
- ❌ Не анімувати змію на фоні. Спокуса велика, результат — 2013 рік.
- ❌ Не ставити градієнти ніде, крім `.gallery`.
- ❌ Не використовувати `--gold` для тексту довшого за рядок. Читати нечитабельно.

## Контрастність (WCAG)

| Пара | Ratio | Статус |
|---|---|---|
| `--text` на `--ink` | 14.8:1 | AAA |
| `--text-body` на `--ink` | 9.2:1 | AAA |
| `--text-mute` на `--ink` | 3.9:1 | AA тільки для 18px+ / bold |
| `--gold` на `--ink` | 8.1:1 | AAA |
| `--ink` на `--gold` | 8.1:1 | AAA — кнопка ок |

`--text-mute` не використовувати для body-тексту 16px. Тільки лейбли, крихти, підписи.
